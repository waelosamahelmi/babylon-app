## Key Visual Features

### 1. **Professional Header**
- Restaurant logo displayed as high-quality dithered image
- Elegant double-line separators (â•)

### 2. **Clear Information Hierarchy**
- Order number prominently displayed with receipt icons
- Time and date with clock icon
- Delivery/pickup status with relevant icons

### 3. **Customer Section**
- Icons for each field (phone, email, location)
- Well-spaced and easy to read
- Address clearly formatted on multiple lines

### 4. **Modern Item List**
- Box-drawing characters for visual separation (â”„)
- Large, bold item names with quantities
- Nested topping display with bullets
- "ILMAINEN" (FREE) clearly marked for included toppings
- Price alignment on the right

### 5. **Summary Section**
- Clean breakdown of costs
- Delivery fee with truck icon
- Total in large, prominent text

### 6. **Interactive QR Code**
- Scannable QR code image
- Clear instructions in Finnish
- Website URL displayed below code

### 7. **Warm Closing**
- Heart icons for emotional connection
- Thank you message in Finnish
- Double borders for visual appeal
